# URSSI Practice

A simple Hello World library with CI and docs.

## Installation

Remote:
```bash
pip install https://github.com/avcopan/urssi-practice
```

Local:
```bash
pip install -e .
```

## Usage

```python
>>> import greetme
>>> greetme.do_it("Mike")
Hey, mike!
```
